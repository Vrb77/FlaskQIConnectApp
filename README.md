
# FlaskQIConnectApp
Deployed Project: https://qiconnecthub.onrender.com
# About Project 🔗 
QiConnect is a web platform that connects customers with vendors.
It allows users to find services, communicate with vendors, and manage service requests efficiently.
# feactures
- Customer has request form and vendor has service form.
- Customer fill the request form by adding problem statement, photos and other details. After submitting form customer can see matching customers who are related to that request. customer can contact to respect vendor or vice versa by messaging using text audio, video, document(just like whatsapp). see the vendor profile. After confirming price and date by vendor. Customer can place order
- customer can edit, close the request. 
- 
- Provide a messaging platform which includes text, audio and video.  
- Best matching vendor and customer will get by their **geolocation** and **matching percentage%**, which can be filter out and search.
